给定一个二叉树 `root` ，返回其最大深度。

二叉树的 **最大深度** 是指从根节点到最远叶子节点的最长路径上的节点数。


**示例 1：**

![1743853143425](image/README/1743853143425.png)

<pre><b>输入：</b>root = [3,9,20,null,null,15,7]
<b>输出：</b>3
</pre>


**示例 2：**

<pre><b>输入：</b>root = [1,null,2]
<b>输出：</b>2
</pre>


**提示：**

* 树中节点的数量在 `[0, 10<sup>4</sup>]` 区间内。
* `-100 <= Node.val <= 100`
